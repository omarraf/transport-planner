# **Sustainability Transport Planner**

An interactive web application that helps users make informed decisions on transportation methods by comparing route distance, duration, carbon dioxide (CO₂) emissions, and weather conditions. The app integrates real-time route calculation and weather forecasts, enabling users to choose the most efficient and sustainable travel options.

---

## **Features**
- 🌍 **Interactive Map**: Visualize routes using OpenStreetMap with draggable start and end markers.
- 🚶‍♀️🚴‍♂️🚗 **Transport Modes**: Compare walking, biking, driving, and public transit options.
- 🌦️ **Weather Insights**: Incorporates live weather data to recommend the most suitable travel mode.
- 📊 **CO₂ Savings Visualization**: Displays emissions and savings compared to driving via intuitive charts.
- 🛠️ **Reset Functionality**: Clear routes, inputs, and results with a single click.

---

## **Technologies Used**
- **Frontend**: HTML, CSS, JavaScript, [Leaflet.js](https://leafletjs.com/) for map rendering.
- **APIs**:
  - [OpenRouteService API](https://openrouteservice.org/) for route calculations.
  - [OpenWeather API](https://openweathermap.org/) for real-time weather forecasts.
- **Visualization**: [Chart.js](https://www.chartjs.org/) for CO₂ savings charts.

---

## **How It Works**
1. Enter the **start** and **end** locations in the input fields.
2. Select your **transport method** (Walking, Biking, Driving, Public Transit).
3. View the calculated **route**, including:
   - Distance
   - Estimated duration
   - CO₂ emissions and savings
   - Weather forecast for the route
4. Use the **reset button** to clear the map and inputs.
